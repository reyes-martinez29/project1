# Guía Visual de Git y GitHub (HTML + CSS)

Una página estática (sin frameworks) que explica Git y GitHub paso a paso, con comandos, ejemplos y buenas prácticas.
La página se construye por secciones usando ramas y Pull Requests para practicar Git de forma real.

## Objetivos de aprendizaje
- Entender el flujo básico: init → add → commit → log → diff
- Aprender ramas, merge y resolución de conflictos
- Conectar con GitHub: clone, push, pull, PRs
- Construir un historial profesional de commits
